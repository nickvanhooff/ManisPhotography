Step 1: Create GitHub OAuth App

  1. Go to: https://github.com/settings/applications/new
  2. Fill in the following details:
Application name: Manis Photography CMS
Homepage URL: https://manis-photography.vercel.app
Authorization callback URL: https://manis-photography.vercel.app/admin/
  • Click "Register application"
  • Copy the Client ID (you'll need this)

    • Step 2: Update Configuration

    Once you have the Client ID, you need to update the config.yml file:

    backend:
      name: github
      repo: nickvanhooff/ManisPhotography
      branch: main
      base_url: https://api.github.com
      auth_type: pkce
      app_id: YOUR_CLIENT_ID_HERE

    Step 3: Test the CMS

    After updating the config:

    1. Go to: https://manis-photography.vercel.app/admin
    2. Click "Login with GitHub"
    3. Authorize the application
    4. You should now have access to the CMS!
